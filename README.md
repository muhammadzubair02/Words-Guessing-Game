# Words-Guessing-Game
An interactive Word Guessing Game built with Python and Tkinter, featuring categorized words (countries, capitals, fruits, vegetables), a 3-level hint system, and a user-friendly GUI. Uses an external JSON file for scalable word management and demonstrates GUI, file handling, and game logic.

Technologies Used:
. Python
. Tkinter (GUI)
. JSON (data storage)

How to Run
1. Clone the repository
2. Make sure `words.json` is in the same folder
3. Run the Python file
